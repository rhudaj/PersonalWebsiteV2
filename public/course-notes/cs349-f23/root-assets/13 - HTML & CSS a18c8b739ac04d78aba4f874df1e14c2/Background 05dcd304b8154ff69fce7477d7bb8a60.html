Default

flex: 0 1 auto
  • Items shrink with the container
  • Container grows to match the items

flex-basis

  • how we tell an element to stick to an ideal size
  • By default, it’s set to auto which means, “Use my height or width.”
  • To make all the elements take up the full space of the parent we can set the child elements to width: 100%, or we can set the flex-basis to 100%, or we can set flex-grow to 1.
  • flex-basis to 1000px or, = “please just try and take up 1000px of space.” If that’s not possible, then the element will take up that much space proportionally to the other elements.
