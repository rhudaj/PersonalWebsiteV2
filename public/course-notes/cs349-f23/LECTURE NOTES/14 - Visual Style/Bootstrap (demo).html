Bootstrap (demo)

Keys

  • Using via CDN links
  • Using bootstrap (no CSS sheets)
    • Baseline typography
    • Ready-to-use grid system
      • responsive, e.g. col-sm means column when size larger than small
    • Widgets
      • buttons, collapse/expand, etc.
    • Icons
      • webfont and svg
