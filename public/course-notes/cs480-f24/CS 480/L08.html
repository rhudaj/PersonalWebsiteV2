Topic: Nonlinear Feature Maps & Kernel Methods

Motivation

Problem:

So far we’ve found linear decision boundaries. But not all data is linearly separable.

Solution:

  1. Map the features to a new feature space
  1. Learn a separating hyperplane there

Feature Maps

A mapping of features, usually to a higher dimension (m>d\small m>dm>d﻿)

ϕ(x):x∈Rd↦Rm\phi (x): x \in \mathbb{R^d} \mapsto \mathbb{R}^mϕ(x):x∈Rd↦Rm﻿

The result is still linear in terms of weights, but non linear with respect to the input features.

How to classify with this approach:

  1. Map your training data with the feature map
  1. Train your model on the mapped set ⇒ w,bw,bw,b﻿
  1. Classify with your model: y^=⟨w,ϕ(x)⟩+b\hat{y}=\langle w, \phi(x) \rangle+by^​=⟨w,ϕ(x)⟩+b﻿

How to choose a feature map?

  • Depends on the problem.
  • Need to know something about structure of data & nature of classification task.
  • e.g. for the “moons task” a polynomial classifier works, but a quadratic does not.

Kernel Functions

A function which computes the inner product of 2 mapped feature vectors as a function of the original vectors.

k(xi,xj)=⟨ϕ(xi),ϕ(xj)⟩k(x_i,x_j)= \langle \phi(x_i), \phi(x_j)\ranglek(xi​,xj​)=⟨ϕ(xi​),ϕ(xj​)⟩﻿

We can also view it as an n×nn \times nn×n﻿ matrix, KijK_{ij}Kij​﻿.

Mercer Kernel

Has 2 specific properties:

  1. Symmetric: k(xi,xj)=k(xj,xi)k(x_i,x_j)=k(x_j,x_i)k(xi​,xj​)=k(xj​,xi​)﻿ || Kij=KjiK_{ij}=K_{ji}Kij​=Kji​﻿
  1. Positive Semi-Definite (kernel matrix)
  • Why the Distinction Matters
    • Symmetry ensures the function behaves consistently no matter the order of inputs.
    • Positive semi-definiteness guarantees that the kernel corresponds to a valid inner product space, which is important for optimization algorithms (like SVM) to work correctly.
