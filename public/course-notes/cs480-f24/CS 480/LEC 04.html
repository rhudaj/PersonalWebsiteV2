Kernel (Parzen) Density Estimation (KDE)

Learning a distribution without a hypothesis class: If you don't have a reference distribution (for the data), use KDE to estimate one.

Idea:

  • Apply a kernel function at observation (point) in the dataset.
    • more points ⇒ high density
  • The probability density for a query point is the average density of all kernels.

k-Nearest Neighbours (k-NN)

Method for classifying points

Idea: For a new query point, if you want to know what class it belongs to, you can look at the k-nearest neighbours and take a majority vote.

Algorithm:

For query point xxx﻿:

  1. Compute the distance between xxx﻿ and every dataset-point.
  1. Find the smallest kkk﻿ distances
  1. Out of these, return the majority class.

def LyodsAlgorithm(D: np.ndarray, k):
    C = None
    # P_C[i] = cluster that D[i] belongs to
    P_C = np.random.choice(k, n)    # Randomly initialize k cluster centroids
    
    new_clusters = True
    while new_clusters:
        new_clusters = False
        # 1) get the clusters, using the P_C map:
        C = [ D[P_C == i] for i in range(k) ]
        # 1) get the new centroids
        cents = [ getClusterCentroid(C_i) for C_i in C ]
        # 2) for each point, assign it to the closest centroid
        for i in range(n):
            closest_centroid_num = np.argmin(
                [ costFunction(D[i], cent) for cent in cents ]
            )
            # check if this is a diff centroid
            if P_C[i] != closest_centroid_num:
                new_clusters = True
                P_C[i] = closest_centroid_num
    # Returns lowest total cost partition
    return C
