Numpy

Axes

  • In a 2-dimensional NumPy array, the axes are the directions along the rows and columns.
  • axis 0 is the “first” axis (down the rows)
  • axis 1 is the second axis (across columns)
  • the axis parameter controls which axis will be collapsed (which one will be aggregated)
    • functions like sum(), mean(), min(), median(), and other statistical functions aggregate your data
    • when you use np.sum() on a 2-d array with the axis parameter, it is going to collapse your 2-d array down to a 1-d array. It will collapse the data and reduce the number of dimensions.

The ptp()function in NumPy obtains the range of values along a given array's axis.
