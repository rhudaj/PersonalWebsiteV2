MLFQ (Multi Level Feedback Queues)

  • Idea

    Give priority to interactive threads (block often)

    Non-interactive threads (preempt often)

  • Example

    Q3Q_3Q3​﻿ is highest, Q1Q_1Q1​﻿ is lowest (3 levels)

    T1,T2T_1,T_2T1​,T2​﻿ arrive

    Both start in Q3Q_3Q3​﻿

    T1T_1T1​﻿ is selected to run

    T1 is preempted.

    T2 can now run.

    A new thread T3 arrives while T2 is
    running

    T2 finished.

    T3 can now run

    T1 can now run.
