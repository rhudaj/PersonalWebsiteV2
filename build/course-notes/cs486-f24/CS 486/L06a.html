Measuring Success

We measure success based on how well the agent performs for new (unseen) data.

Bias

  • Tendency to prefer one hypothesis over the other.
  • Necessary for predicting on unseen data using an inductive process.
  • What makes a bias “good” = how it performs in real-life
  • N, P are extreme biases

Metrics to measure a binary-classification-model’s performance

Different types of errors (wrong predictions) have different consequences.

  • e.g.

    predicting if a patient has a disease:

    Actual PredictedPredicted

    The answer to this dictates how we should bias our model.
