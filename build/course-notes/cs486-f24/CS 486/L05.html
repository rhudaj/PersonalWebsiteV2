Valid Argument

AAA﻿ is a valid argument if:

  1. premises ⊨\models⊨﻿ conclusions (A evaluated)
  1. conclusions (A evaluated) are true in every model of the premises
  1. Premises are never all true (no model) and conclusions are false

    * If premises are inconsistent, can deduce anything

  1. AAA﻿ →\rarr→﻿ conclusions, is always true (tautology)

Proofs

Knowledge base: set of axioms (just statements)

Proof procedure: way of proving theorems (deductions)

  • KB⊢gKB\vdash gKB⊢g﻿ :
    • ggg﻿ can be derived from kb using proof procedure. ggg﻿ is now a theorem

Sound proof

  • Every theorem we prove is a logical consequence
  • KB⊢g  ⟹  KB⊨gKB \vdash g \implies KB \models gKB⊢g⟹KB⊨g﻿

Complete Proof:

  • Every logical consequence can be proved
  • KB⊨g  ⟹  KB⊢gKB\models g \implies KB \vdash gKB⊨g⟹KB⊢g﻿

Drawback of a Formal System

A human can always infer something about it, that it cannot infer about itself.

Planning

(Think of it like: logic | graph search)

Given:

  1. Actions (inference rules | arcs)
  1. Initial state (axioms | start node)
  1. Goal (deduction | goal node)

Find:

Sequence of actions to solve a goal (proof | path)

(Deterministic) Planner

Planning for deterministic world (described by logic)

  • Assumptions
    1. Single agent
    1. Deterministic world
    1. No exogenous events (outside control of agent)
    1. Agent knows its state (fully observable)
    1. Time progresses discretely, state to state
    1. Goals are predicates of states (not complex)
  • Action = partial function
    • “partial” — actions only allowed in certain states based on preconditions

State Based Representation (of Planner & Actions)

  • Planning: moving between states towards goal
  • state: set of features that represent the world.
  • preconditions: are propositions of features
  • For each feature:
    • causal rules: when the feature changes
    • frame rules: when the feature does not change.
  • Notation
    • features: capitilize
    • values of features: lower case
    • If XXX﻿ is a feature, X′X'X′﻿ is the feature after some action.
