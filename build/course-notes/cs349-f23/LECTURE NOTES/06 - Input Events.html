FundamentalEvent

export interface FundamentalEvent {
  type:
    | "resize"
    | "keydown"
    | "keyup"
    | "mousemove"
    | "mousedown"
    | "mouseup"
    | "null";
  timeStamp: DOMHighResTimeStamp;
  width?: number;
  height?: number;
  x?: number;
  y?: number;
  key?: string;
}
  • Why null?
    • some translators are triggered only by time (not event)
      • solution: send a “null” event when no other events
      • if there are events, the translators can use time of those instead
