18 - Direct Manipulation

Interaction Model

  • a way to describe an interface
  • how to combine interaction techniques
  • we want an interaction model that works with typical desktop and is useful to describe direct manipulation
    • eg: instrumental interaction

instrumental interaction

model of interaction

  • on how we naturally use tools to manipulate objects in the real world
  • interface has interaction instruments and domain object
    • domain object = data of interest, manipulated with instrument
    • (text is the domain, instrument: bold, italics, select all, indent)

Instrument Activation

how instrument is triggered for use

UI layout and design is concerned with the tradeoff of these costs

  • ie: can‚Äôt have all the tools laid out (all spatial) but can‚Äôt have them all hidden (all temporal)
  1. spatially

    Has a movement cost

    • click a button
    • scroll-bar
  1. Temporal

    Has a activation time cost

    • Note: temporal is about ‚Äúactivation time‚Äù. So for a slider. You don‚Äôt count the time it takes to slide it, only to activate the slider (which takes no time).
    • draw rectangle, have to select rectangle from drop-down, then select what type of rectangle, then you can draw them spatially. There is a time-cost.

Note: almost all elements are spatial (because in a GUI you have to click to use)

button: spatial

scroll-bar: spatial

rotate: temporal

scale: temporal

translate: spatial

Reification

Turning a concept into something real

  • ie: instrument is a reification of a command
  • e.g: scrollbar reifies scroll commands

Meta Instrument

Instrument acting on another instrument.

  • object of interest = other instrument
  • ie: in photoshop, increasing the brush size
  • ie: instrument = fill tool, but then having a color-wheel is changing what the fill tool does.

Instrument Properties

3 Degrees

  • Indirection
  • Integration
  • Compatibility

Indirection

distance from instrument to domain object *WHEN YOUR USING IT*

2 dimensions:

  1. Spatial (how far to move mouse to activate)

    eg: Near: drag to translate, handles to resize

    eg: Medium: scrollbar near page

    eg: Far: dialog boxes

  1. Temporal (how long to wait to activate)

    eg: short: direct dragging response

    eg: med: activate tool in toolbar

    eg: long: using dialog, finishing drag and drop
