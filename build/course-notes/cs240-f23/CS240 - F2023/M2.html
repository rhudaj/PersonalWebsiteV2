Abstract Data Types

Description of info & collection of operations on that info

Many realizations of an ADT

  • how the info is stored → data structure
  • how operations are performed → algorithms

Stack

info - Collection of Items. Last in first out.

operations: push, pop

realization - arrays, linked-lists, etc…

Queue

info - Collection of Items. First in, first out

operations: enqueue, dequeue

realization - circular arrays, linked-lists, etc

Priority Queue

Generalizes Stack & Queue

  • Stack: Inserted items get highest priority, delete-max
  • Queue: Inserted items get lowest priority, delete max

Info - collection of items <priority, value>

Operations: insert, delete-(max|min)

Realizations

  • 1 - Unsorted Array

    insert - add to end of array, O(1)O(1)O(1)﻿

    delete-max - search entire array, O(n)O(n)O(n)﻿

    PQ-Sort - yields selection-sort

  • 2 - Sorted Array

    insert - sort the array, O(?)O(?)O(?)﻿

    delete-max - delete last item, O(1)O(1)O(1)﻿

    PQ-Sort - yields insertion-sort
