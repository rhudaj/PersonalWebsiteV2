HAVING

  • SYNTAX
    SELECT [<other-attrs>] [AGG(<attrs>)]
...
GROUP BY [<other-attrs>]
HAVING <condition>
  • EX

    publications with more than one author

    SELECT publication, count(author) as cnt
FROM wrote
GROUP BY publication
HAVING count(author) > 1

Notes:

  • WHERE cannot be used with aggregate functions
  • Apply a filter on the result of GROUP BY (filtering groups)
    • The HAVING clause is applied to each group of the grouped table. If there is no GROUP BY clause, the HAVING clause is applied to the entire result as a single group.
  • Must reference groups (whatever shows up in the result of group by)
  • The aggregate functions used in the HAVING clause may be different from those in the SELECT clause. The grouping is common.
