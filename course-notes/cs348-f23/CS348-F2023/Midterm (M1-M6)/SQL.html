SQL

(data definition language)

(data manipulation language)

Overview

  • Interface for the Relational Model
  • Core Features:
    • integrity constraints
    • a security model
    • Declarative DML
    • ACID Transactions
  • Additional Features:
    • BAG (multiset) semantics
    • NULL value

Data Types

  • integer (32 bit)
  • smallint (16 bit)
  • decimal(m,n) (fixed decimal)
  • float (32 bit)
  • char(n) (length n)
  • varchar(n)(at most n)
  • date (year/month/day)
  • time (hh:mm:ss:ss)
