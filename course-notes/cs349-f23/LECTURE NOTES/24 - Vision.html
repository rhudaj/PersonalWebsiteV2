Computer vision

  • algorithms that allow computers to make intelligent assertions about digital images and video
  • ‚Äúfor Human Computer Interaction‚Äù is when a computer is controlled using a video camera

Basic Algorithm

for each video frame:

  • identify a "special" pixel based on some criteria
  • use that pixel's location to control mouse position

  • ie: identify target HSL (ie: pink), and then search for the closest HSL value to it

Alternate Algorithm

for each video frame:

  • identify the count of some special pixel
  • use that count to control the computer

  • ie: identify a range of acceptable HSL values, count in that range

More Complex 1

for each video frame:

  • subtract this frame from last frame
  • use differences to estimate how far each pixel moved
  • use total pixel movement to control the computer

More Complex 2

for each video frame:

  • subtract out "background pixels"
  • find "blobs" of connected pixels
  • use blob position and/or shape to control the computer

More Complex 3

for each video frame:

  • send frame to deep learning model to recognize "features"
  • use features to control the computer
