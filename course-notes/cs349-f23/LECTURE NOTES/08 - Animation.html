2. Timer

  • Needs:
    • a duration
    • a start-time
    • an update(time) function to check if timer is finished
    • a isRunning() to check if timer is running
    • optional: a callback() → when timer is finished

3. Tweening

Basic Animation Object

  • construct with animation parameters
    • startValue
    • endValue
    • duration
    • startTime
  • start it with current time
  • updateValue()
    • animation callback
    • updates animation value with time

Easing Functions

Easing Function:

  • Making time a function of something else
value = Lerp(startVal, endVal, t)
value = Lerp(startVal, endVal, easing(t))

Types 

//TEMPLATE
type EasingFunction = (t:number) => number;

// COMMON FUNCTIONS

const flip = (t) => 1 - t

const easeOut = (t) => Math.pow(t-2)

const easeIn = (t) => flip(easeOut(flip(t))

const easeInOut = (t) => (lerp(easeOut(t), easeIn(t), t)

Animation Manager

  • provide by some toolkits
  • keeps a list of active animations
  • updates each animation every frame
  • remove animations when they finish

4. Key-Framing

Array of KeyFrame's

  • {time, targetValue} 
// Find keyframe i and keyframe i+1 for the current time

(time > keyframe[i].time) && (time < keyframe[i+1].time)

//Tween value i & i+1

Ex 1

When time = 800

Tween keyframe[0] & keyframe[1]

t = (800 - 0) / (1000 - 0)
value = 400 + t * (50-400)

Ex 2

When time = 3000

Tween keyframe[2] & keyframe[3]

t = (3000 - 2500) / (5000-2500)
value = 400+t * (50 - 400)
