2. Route Construction

path from root → target

  • Code
    function buildTargetRoute(mx, my, target: SKElement) {
	route = []
	// RECURSIVE CASE -> SUBTREE
	if(target instanceof SKContainer) {
		target.children.forEach((child) =>
			buildTargetRoute(mx - target.x, my - target.y, child)
	}
	// BASE CASE -> This was the target that was hit
	if(element.hittest(mx,my) return [element, ...route]
	else return route
}

3. Event Propagation

Event binding: adding/calling a handler to a widget for specific events

2 Phases

  • Events go up and down, widget can use the event either time with a handler
  • The handler can also stop propagation
    • capture handler → stop rest of capture & bubble phase (return true)
  • Phase 1 — capture
    • down the route
  • Phase 2 — bubble
    • up through parents to root

Positional Dispatch - Limitations

Sends ALL mouse events to the widget that’s underneath the mouse

  • ie: you go off the scroll bar, now the adjacent widget get’s the mouse event

  • solution → consider which widget is in FOCUS

Focus Dispatch

Mouse Focus

  • mouseDown → focus
  • mouseUp → remove focus

KeyboardFocus

  • whatever text field was last clicked
